* 包
   *  初始化 
      *  npm init
     
              package name    包名称
              version         版本
              description     描述备注
              entry point     入口       默认(index)
              test command    测试命令
              git repository  远程地址
              keyword         关键词
              author          作者
              license         开源协议    MIT

   * 包下载
     * npm i package name 包名称   
      
            -g 全局 globe

  * 包查看
    * npm ls    （list
  
  * 包卸载 
    * npm uninstall 包名
   
  
  * 包更新 
    * npm update 包名
